# CSS Cheatsheet
## Units
### Colors
- Hex colors
  - #rgb(a): `#DEAD`
  - #rrggbb(aa): `#BEEFACE`
- rgba(red_channel, green_channel, blue_channel, alpha_channel): `rgba(255,0,234,100)`
- Color names (red, green, blue, white, gray, etc.): `red`
### Measurements
- Absolute units
  - pixels: `1337px`
- Relative units
  - percentage: `69%`
  - em: `404em`
  - rem: `420rem`
